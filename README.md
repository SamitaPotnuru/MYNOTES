# MYNOTES

A **mini Notion-style Notes App** built with **HTML, CSS, and JavaScript**.  
This app allows users to **add, edit, delete, and update notes in real-time**. Notes are saved in the browser using `localStorage`, so they persist across sessions.

## 📝 Features

- **Add Notes:** Create new notes instantly with a single click.  
- **Edit Notes:** Type in the note editor and see changes in real-time.  
- **Delete Notes:** Remove notes easily with a confirmation prompt.  
- **Persistent Storage:** Notes are saved locally using `localStorage`.  
- **Responsive UI:** Clean, modern design, works on desktop and mobile.  
- **User-Friendly:** Sidebar navigation with active note highlighting.

## 💻 Technologies Used

- HTML5  
- CSS3 (Flexbox & Grid)  
- JavaScript (Vanilla JS)  
- `localStorage` for data persistence
